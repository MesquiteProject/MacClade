Recently-added features

There were a few new features added to versions 3.01 to 3.05. These are:

Added to 3.01:

  • Footnotes can now be included when exporting a simple table file.

Added to 3.02:

  • Display options for Trace All Changes have changed substantially. Most importantly, you can now display trees in the tree window with branch length proportional to the number of reconstructed changes. Also, you can choose to have tick marks unlabeled.
  • A text file of chart calculations (p. 275) can now be written for State Changes + Stasis charts (pp. 287&endash;293) calculated over a single tree if minimum-maximum changes are requested.

Added to 3.03:

  • NEXUS blocks can now be terminated with either END or ENDBLOCK.

Added to 3.04:

  • MacClade can speak the cells in the matrix; this facility is useful for proofreading.

Added to 3.05:

  • Export to MEGA format
  • More predefined genetic codes
  • Increase in number of allowed continuous characters to 10
  • Compress Characters now available (will search for characters with identical state distributions and of identical type, and merge all such characters together into one; the weight of the compressed character is the sum of weights of the included characters).

Details about these features are provided in the Supplement to the MacClade Book .
